COO Management VBA

Sistem manajemen data transaksi Certificate of Origin (COO) berbasis Microsoft Excel dengan Visual Basic for Applications (VBA), dikembangkan untuk Kamar Dagang dan Industri (KADIN) Provinsi Jawa Timur.

📌 Latar Belakang

Pengelolaan data transaksi pengiriman ekspor untuk keperluan penerbitan Certificate of Origin (COO) di KADIN Jawa Timur sebelumnya dilakukan secara manual, sehingga rentan terhadap inkonsistensi format pencatatan, kesulitan penelusuran data, dan risiko kesalahan input. Proyek ini dikembangkan sebagai solusi optimalisasi melalui produk digital berbasis Excel VBA yang terstruktur.

Proyek ini merupakan hasil mini proyek mahasiswa Program Studi S1 Sistem Informasi, Fakultas Teknik, Universitas Negeri Surabaya, dalam program Mobilitas Akademik (Magang MBKM) di KADIN Jawa Timur periode Februari–Mei 2026.

✨ Fitur

Form Input Interaktif (UserForm) — antarmuka input data transaksi pengiriman ekspor yang terstruktur
Fungsi Save — menyimpan data secara otomatis ke dalam Excel Table
Fungsi Delete — menghapus data transaksi yang dipilih
Fungsi Clear — mengosongkan form sebelum input data baru
Fitur Search — pencarian data berbasis kriteria menggunakan Advanced Filter Excel
Fitur Calendar (Date Picker) — mempermudah input tanggal melalui kalender interaktif
Tampilan ListBox — menampilkan hasil data secara langsung di dalam form
